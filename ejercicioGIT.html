  1. ¿Que es un sistema de control de versiones?

    Es una forma de poder poner en común varias versiones o partes diferentes de un proyecto para posteriormente fusionarlas o mejorarlas.

  2. ¿Cual es la diferencia entre git y github?

    Git es un sistema de control de versiones el cual tiene sus comandos predeterminados con los que se pueden realizar diferentes acciones con los proyectos mientras que GitHub es una plataforma donde se pueden subir o modificar los proyectos en repositorios usando Git.

  3. ¿Cómo creo una rama en git?

    Para crear una nueva rama hay que usar la orden "git branch nombre" indicando el nombre que queremos darle a la rama.
